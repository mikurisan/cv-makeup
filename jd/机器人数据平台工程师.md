## 公司

上海它石智航技术有限公司

## 职责

1. 负责机器人数据闭环平台的建设与持续迭代，覆盖数据采集、处理、标注、质检、数据集构建、版本管理与效果回流等核心环节。需要把分散的数据流程逐步沉淀为标准化、可复用、可追踪的平台能力，支撑算法和研发团队高效使用数据。

2. 负责图像、视频、日志、传感器等多源异构数据的处理链路开发，完成解析、清洗、切片、抽帧、过滤、结构化等关键流程。需要持续优化链路的吞吐、稳定性和可观测性，保障大规模数据任务能够稳定运行。

3. 搭建数据工程相关服务与工具，包括数据任务编排、样本筛选、统计分析、质量评估、异常数据发现等能力。通过平台化方式提升数据生产效率，减少重复性人工操作，帮助团队更快完成数据构建与问题定位。

4. 结合 AI Agent / LLM 能力，推动数据处理、规则生成、任务执行、质检分析等场景的自动化落地。需要将部分专家经验抽象为可复用的 tool、skill 或 workflow，让系统能够在约束下完成更复杂的数据闭环任务。

5. 与算法、产品及业务团队协作，理解模型或策略迭代中的数据需求，并支持相关问题的分析、定位和闭环优化。需要从业务目标反推数据方案，推动数据平台真正服务于模型效果提升和研发效率提升。

6. 负责数据平台相关服务的运维与性能优化，完善日志、监控、告警、任务追踪和技术文档建设。需要持续解决数据链路中的异常、性能瓶颈和系统稳定性问题，提升平台整体可用性和工程质量。

## 任职要求

1、本科及以上学历，计算机、软件工程、自动化、人工智能等相关专业，具备 2 年及以上后端开发、数据工程或平台开发经验。希望候选人有较强的工程落地能力，而不只是停留在方案设计或单点脚本开发层面。

2、精通 Python，具备扎实的工程编码能力，能够独立完成后端服务、数据处理脚本和内部工具的开发与维护。熟悉 Linux 开发环境，具备良好的代码规范、调试能力和基本的软件工程意识。

3、熟悉常见数据处理与任务调度技术，有 ETL、数据流水线、批处理或工作流系统开发经验。能够理解大规模数据任务从输入、处理、产出到监控追踪的完整过程，并具备实际问题排查与优化经验。

4、理解数据闭环相关流程，对数据清洗、样本构建、质量评估、版本管理、问题回流等环节有实际项目经验。能够站在平台视角理解"数据如何支持模型或策略迭代"，而不是只关注单一处理任务本身。

5、具备良好的问题排查能力、跨团队沟通能力和工程协作意识，能够独立推进复杂问题的分析与落地。有机器人、自动驾驶、智能硬件或复杂数据平台背景者优先，对非结构化数据场景有经验会更匹配。

加分项

1、有模型训练平台、训练数据管线或模型评测体系建设经验，理解训练任务从数据准备、任务执行到效果验证的基本流程。

2、熟悉大规模并行计算、分布式训练或 GPU 集群相关技术，如 DDP、Ray、Kubernetes、任务调度与资源管理等。能够理解训练系统在资源分配、任务编排、吞吐优化和稳定性保障上的核心问题者优先。

3、有机器人、自动驾驶、具身智能等领域的数据平台或数据闭环系统经验，熟悉多传感器数据、时序数据或复杂日志数据处理场景。对真实业务中的数据生产链路、问题归因与迭代节奏有经验，会更快进入角色。

4、熟悉 Docker、K8s、CI/CD、对象存储、消息队列等基础设施技术，能够支持平台服务的部署、维护与持续迭代。

5、有将 AI Agent 应用于数据系统、研发流程或训练流程自动化的实际落地经验，能够从"概念验证"走到"业务可用"。
